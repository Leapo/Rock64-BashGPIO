# Rock64-Utilities
Libraries and utilities for the Rock64 single-board computer

# Bash Libraries
rock64-gpiolib
A basic GPIO library for the Rock64

* gpio-setup
* gpio-output
* gpio-input
* gpio-pwm
  - gpio-pwm-calc
  - gpio-pwm-raw
* gpio-cleanup

# Python Libraries
R64.GPIO
A re-implementation of the RPi.GPIO library for the Rock64. Currently under development.
